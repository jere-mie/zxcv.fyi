# zxcv.fyi

The hub site for the zxcv.fyi collection of free, open-source projects.

**Live:** https://zxcv.fyi

## Tech

Plain HTML, CSS, and a small amount of vanilla JS. No frameworks, no build step, no dependencies.
Hosted via GitHub Pages.

## Development

Just open `index.html` in a browser - no build process required.
